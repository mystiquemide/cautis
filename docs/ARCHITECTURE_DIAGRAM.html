Cautis — Guardian Protocol Architecture

On-chain AI agent monitoring and real-time anomaly detection on Casper Network

Contract Entry Points

  • register_agent — stake min 10 CSPR
  • freeze_agent — owner-only lockdown
  • unfreeze_agent — owner-only restore
  • stake — add security deposit
  • set_daily_limit — per-agent caps
  • get_agent / is_frozen — read-only

Guardian Decision Flow

  • Agent submits tx via Cautis API
  • Guardian AI (Claude) analyzes context
  • Hard rules: daily limit exceeded
  • Pattern analysis: anomaly detection
  • Decision: allow | flag | freeze
  • Freeze → on-chain contract call

Infrastructure

  • Smart Contract: Odra 2.8.1 · Rust
  • Backend: Next.js 16 · Edge Runtime
  • AI: Claude Sonnet 4 · Anthropic
  • Data: CSPR.cloud Indexer
  • Payments: x402 Protocol
  • Network: Casper Testnet
